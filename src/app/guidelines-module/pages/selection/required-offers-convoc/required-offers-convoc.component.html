save GUARDAR Y PUBLICAR OFERTAS Y CONVOCATORIAS

ADMIN

1 - Creo una oferta/convocatoria nueva

  • 1.1 -> ¿Cuando la puedo guardar?

    • Para guardar una oferta/convocatoria nueva el único campo obligatorio es el título.
    • Una oferta así estará "incompleta" y no podrá publicarse

  • 1.2 -> ¿Cuándo la puedo publicar?

    # Ofertas

    • Los campos obligatorios para publicar son:
      • Posición/puesto
      • Categoría portal
      • Tipo
      • Plazas *Solo para ofertas Públicas y Privadas.
      • Sueldo
      • Fecha de publicación
      • Fecha de Vencimiento
      • Descripción
    • Los campos opcionales para publicar son:
      • Baremación
      • Fases
      • Documentación
      • Convocatoria enlazada
      • Tasas

    # Convocatorias

    • Los campos obligatorios para publicar son:
      • Categoría portal
      • Tipo
      • Fecha de publicación
      • Fecha de Vencimiento
      • Descripción
      • Que tenga al menos una oferta enlazada
    • No hay campos opcionales para publicar una convocatoria

2 - Edito una oferta/convocatoria

Todo descrito en la creación aplica a la edición. Por ejemplo, si una oferta publicada se edita y se borra el sueldo o las fechas de publicación o vencimiento no encajan con la fecha actual pasará a estar incompleta.
