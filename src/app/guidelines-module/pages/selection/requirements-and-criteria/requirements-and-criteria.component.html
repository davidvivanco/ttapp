Qué son y cómo funcionan

1 - Descripción

Un baremo es un conjunto de criterios que a su vez es un conjunto de valoraciones o combinaciones utilizados para valorar/puntuar a un usuario en un proceso de selección.

2 - Funcional

  • 2.1 - ¿Qué necesitamos para poder crear baremos?
    • Para poder crear baremos debe haber un schema de cv publicado.
    • Campos obligatorios para crear un baremo:
      • Título
    • Campos opcionales para crear un baremo:
      • Descripción
  • 2.2 - ¿Qué necesitamos para poder usar baremos?
    • Para poder usar baremos deben tener al menos 1 criterio y el criterio al menos 1 valoración o que todos los méritos puntúen.
  • 2.3 - Añadir Criterios a un baremo
    • 2.3.1 - ¿Qué es un criterio?
      • Un criterio define el bloque de CV a puntuar y la fórmula que se empleará para puntuar los elementos de dicho bloque.
      • Puede haber más de un criterio para el mismo bloque dentro del mismo baremo pero con fórmulas distintas.
    • 2.3.2 - Campos obligatorios para crear un criterio
      • Título
      • Bloque de CV al que hace referencia
      • Fórmula a utilizar. En la fórmula habrá que decidir si todos los méritos puntúan o no. Si todos los méritos puntúan no hace falta añadir valoraciones/combinaciones ya que cualquiera puntuará.
    • 2.3.3 - Fórmula a usar de un criterio
      • La fórmula define como puntuaremos. Puntúa si existe una combinación o condicionales específicos (solo tenemos estas 2 hoy por hoy)
      • La fórmula tiene:
        • Si todos lo méritos puntúan. Al elegir esta opción cualquier combinación de elementos de este bloque de CV punturá. Y todos lo harán por igual (el admin tiene que definir la puntuación por mérito aquí)
        • Puntuación máxima global. La puntuación máxima que un usuario podrá obtener en este bloque de CV usando este criterio
    • 2.3.4 - Campos obligatorios para poder usar un criterio
      • Los anteriores
      • Y al menos 1 valoración o que todos los méritos puntúen
    • 2.3.5 - Añadir Valoraciones a un criterio
      • Una valoración es la combinación de elementos de un bloque de cv que otorgará N puntos al usuario.
      • Por ejemplo, si el bloque "idiomas" tiene un select de idiomas, otro de nivel y otro de escuela la valoración o combinación sería:
        inglés + nivel 1 + Cambridge
      • A esa combinación se le aplicará la fórmula elegida de puntuación al baremar a un usuario en un proceso de selección y se le asignará una puntuación.
  • 2.4 - Añadir Otros Criterios a un baremo
    • 2.4.1 - ¿Qué son otros criterios?
      • Son criterios específicos de puntuación que dependen de cada cliente.
      • La colección en bbdd es "otherMerits"
